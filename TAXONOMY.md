# Darling, What If — Taxonomy & Tagging Guide

The single source of truth for how posts are organized. Pick from this file every time
you publish. Never invent tags on the fly — that's what created the original 132-tag mess.

---

## The Golden Rule

> **1 category + 2–4 tags (aim for 3). Every tag comes from the controlled list below.**

- **Category** = the post's one true home (the badge, breadcrumb, URL parent, and footer list).
- **Tags** = 2–4 reusable sub-topics, all chosen from this file. Tag chips link to the post's category page.
- A post belongs to **exactly one** category. If it overlaps another, *cross-link* it (feature it on the other category page or in a related-posts block) — do **not** give it a second category badge.

---

## Publishing checklist (per post)

1. Pick **1 category** (of the 8).
2. Pick **2–4 tags** from that category's controlled list (3 is the sweet spot).
3. Make sure each tag chip links to the **category page** (the template already does this).
4. If it spans another category, **cross-link** instead of re-categorizing.
5. Put one-off, post-specific phrases in the **title / meta description**, NOT in tags.

---

## Tag vs. keyword — the test

Ask: *"Would this exact tag plausibly appear on ~3+ other posts over time?"*

- **Yes** → it's a real **tag**. Use it from the list.
- **No** (it only fits this one post, e.g. `melanin rich foods for hair`) → it's a **keyword**. Put it in the `<title>` and meta description, not the tag chips.

## Folding old tags into the list (consolidation key)

When you hit an old/messy tag, map it to its controlled tag instead of keeping it:

- `Mental Health`, `mental exhaustion`, `quiet anxiety`, `ambient anxiety` → **anxiety** or **nervous system**
- `gentle living`, `slow living`, `romanticize your life` → **soft living**
- `morning pages`, `daily brain dump`, `journaling for mental health` → **journaling**
- `emotional intimacy`, `emotional connection`, `attunement`, `being seen` → **emotional safety**
- `grief and love`, `losing a partner`, `healing after loss` → **healing after loss**
- `Macro Nutrition`, `High Protein Diet`, `90-30-50 Method` → **high protein**
- `Period Health/Cramps/Cravings`, `Menstrual Nutrition` → **cycle syncing**
- `microbiome`, `digestion`, `apple cider vinegar`, `kitchen remedies` → **gut health** / **microbiome**
- `Blood Sugar Balance`, `Sugar Cravings`, `Magnesium Deficiency` → **blood sugar balance**
- `barrier repair`, `retinol`, `over-exfoliation` → **skin barrier**
- `SPF`, `blue light skin`, `screen time skincare` → **SPF & anti-aging**
- `foods that prevent grey hair`, `hair nutrition`, `B12 for hair` → **hair health**
- `dogs`, `companionship`, `dog love` → **dog mom** / **the human-dog bond**
- `things to do in Maui`, `Maui travel guide`, `island life mindset` → **Maui** / **slow travel**

Anything that doesn't fit a controlled tag is almost always a **keyword** → move it to the title/meta.

---

## Governance

- **Adding a new tag:** only when a genuinely new theme has appeared on **~3+ posts**. Then add it to the right category here first, *then* use it. One post ≠ a new tag.
- **Quarterly 10-min tidy:** scan for any stray one-off tags that slipped in; map them into an existing tag or promote per the rule above.
- **Keep this file in the repo.** It is the source of truth — update it here before changing how you tag.
